PiC MAP - A Web Application for Managing Projects

Link to Application
The application provides a platform for users to manage their projects. Users can register an account and sign in to create new and view their current projects, as well as make new projects. Within each project, users can add requirements, issues, and/or tasks, and assign users that are on the project to each. Each project also includes a chatroom in which users in the project can communicate.
Analysis of Different Algorithms for Generating Prime Numbers
This console application lets the user generate an input number of prime numbers using one of a few algorithms. In particular, I explored an algorithm that tries to quicken a brute force algorithm for determining the primality of an integer N (checking each integer from 1 to N as a factor of N). The algorithm relies on the fact that you need only check prime numbers less than N, as checking composite numbers is redundant.
Queue Application
This console application provides a simulation for customers in a queue. The user can first input information for the number of minutes for the simulation to run, as well as information about the customers entering the queue. The application includes a graphical display of the current state of the queue.
